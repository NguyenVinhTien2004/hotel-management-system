# Hình ảnh Phòng Đơn

Thư mục này chứa hình ảnh cho loại phòng đơn.

## Quy tắc đặt tên file:
- Sử dụng định dạng: `phong-don-{số}.jpg` hoặc `phong-don-{số}.png`
- Ví dụ: `phong-don-1.jpg`, `phong-don-2.jpg`

## Kích thước khuyến nghị:
- Độ phân giải: 800x600px hoặc cao hơn
- Tỷ lệ khung hình: 4:3 hoặc 16:9
