# Hình ảnh Phòng VIP

Thư mục này chứa hình ảnh cho loại phòng VIP.

## Quy tắc đặt tên file:
- Sử dụng định dạng: `phong-vip-{số}.jpg` hoặc `phong-vip-{số}.png`
- Ví dụ: `phong-vip-1.jpg`, `phong-vip-2.jpg`

## Kích thước khuyến nghị:
- Độ phân giải: 800x600px hoặc cao hơn
- Tỷ lệ khung hình: 4:3 hoặc 16:9
