# Hình ảnh Phòng Gia Đình

Thư mục này chứa hình ảnh cho loại phòng gia đình.

## Quy tắc đặt tên file:
- Sử dụng định dạng: `phong-gia-dinh-{số}.jpg` hoặc `phong-gia-dinh-{số}.png`
- Ví dụ: `phong-gia-dinh-1.jpg`, `phong-gia-dinh-2.jpg`

## Kích thước khuyến nghị:
- Độ phân giải: 800x600px hoặc cao hơn
- Tỷ lệ khung hình: 4:3 hoặc 16:9
