# Hình ảnh Phòng Đôi

Thư mục này chứa hình ảnh cho loại phòng đôi.

## Quy tắc đặt tên file:
- Sử dụng định dạng: `phong-doi-{số}.jpg` hoặc `phong-doi-{số}.png`
- Ví dụ: `phong-doi-1.jpg`, `phong-doi-2.jpg`

## Kích thước khuyến nghị:
- Độ phân giải: 800x600px hoặc cao hơn
- Tỷ lệ khung hình: 4:3 hoặc 16:9
