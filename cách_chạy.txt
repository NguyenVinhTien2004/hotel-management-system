=== HƯỚNG DẪN CHẠY ỨNG DỤNG ===

*** SỬA LỖI CACHE (khi đặt phòng không tự động cập nhật) ***
1. Nhấn F12 trong trình duyệt
2. Chuột phải vào nút Refresh
3. Chọn "Empty Cache and Hard Reload"
4. HOẶC nhấn Ctrl + Shift + R
5. Restart server: Ctrl+C trong terminal backend, rồi chạy lại "node server.js"

1. SETUP DATABASE (chỉ chạy lần đầu):
   - Mở MySQL và tạo database: CREATE DATABASE quanlykhachsan;
   - Chạy: mysql -u root -p quanlykhachsan < backend\database_schema.sql
   - Chạy: mysql -u root -p quanlykhachsan < backend\reset_rooms_20.sql
   - Chạy: mysql -u root -p quanlykhachsan < fix_room_prices.sql
   - Chạy: mysql -u root -p quanlykhachsan < add_service_ratings_table.sql
   - Chạy: mysql -u root -p quanlykhachsan < fix_service_prices.sql
   - Chạy: mysql -u root -p quanlykhachsan < clean_price_data.sql
   - (Tùy chọn) Chạy: mysql -u root -p quanlykhachsan < format_price_view.sql

2. CHẠY SERVER:
   - cd backend (nếu terminal chưa ở thư mục backend)
   - node server.js

3. CHẠY FRONTEND:
   - Mở trình duyệt và truy cập: http://localhost:3001
   - HOẶC click chuột phải vào file index.html và chọn "Open with Live Server"
   - Lưu ý: Frontend được serve trực tiếp từ backend server (port 3001)
   
   LƯU Ý: Sử dụng thư mục frontend ở gốc (đã sửa lỗi), không phải frontend-20251108T081940Z-1-001
